# ValuationVantage

This repository contains the website for ValuationVantage, a specialized lead generation system for valuation professionals.

## Website Structure

- `index.html` - Main landing page
- `thanks.html` - Thank you page shown after form submission
- `CNAME` - Custom domain configuration for GitHub Pages

## Setup Instructions

1. This site is hosted using GitHub Pages with a custom domain
2. The form uses Formspree for handling submissions
3. Replace the Formspree form ID in index.html with your own after creating an account
